AUTOR: ROBERTO PORTILLO TORRES

Para ejecutar compilar el fichero main.cpp.

El ejecutable acepta un entero como parametro. Introducir  1 o 2 
para realizar la búsqueda sobre el grafo predefinido seleccionado.

En caso de no introducir parametro se pedira un número de nodos, 
se generará un grafo aleatorio de dicho número de nodos y se ejecutará
el algoritmo sobre él.
